Challenge Selected for Group 1 (4.23)

Challenge:

Consider a two-player game played with pegs of six different colors. One player (the encrypter) selects four pegs of different colors and places them in a certain order. The other player (the decrypter) guesses the order to uncover the hidden code, guided by feedback using black and white pegs. The challenge involves writing a JavaScript program that simulates this encryption-decryption game, enhancing logical thinking and problem-solving skills.

Justification:

This challenge was an excellent choice for Group 1 as it required the integration of complex logic and strategic thinking, skills essential for computer science students. By designing and implementing a game that simulates cryptographic principles, participants improved their programming prowess and problem-solving capabilities. This experience not only deepened their understanding of JavaScript but also fostered a practical application of encryption and decryption methodologies, which are pivotal in real-world computer science scenarios.

Group 10 Assigned Problem (3.25) Understanding:

The CSS 'fixed' positioning is a powerful feature that anchors an element to a specific position relative to the viewport, regardless of scrolling. This property is particularly useful for creating persistently visible elements like navigation bars or floating action buttons. Testing across browsers reveals its broad support, although issues may arise in older versions like IE6, highlighting the importance of cross-browser compatibility testing.

How we completed the Challenge

To address the challenge, we applied 'fixed' positioning to the navbar of our website, demonstrating a practical use of CSS positioning. This technique allowed us to illustrate the contrasting behaviors between 'static' and 'fixed' positioning. With 'fixed' positioning, the navbar remains consistently visible at the top of the viewport as users scroll through the page, enhancing usability and accessibility. In contrast, 'static' positioning would leave the navbar at its original location in the document flow, disappearing from view as the user scrolls down. This experiment not only met the challenge requirements but also showcased how strategic CSS styling can influence user experience and interface dynamics.

How Group 1 completed the Challenge

Group 1 developed The Time Bomb Defusal game is an engaging and interactive project that combines problem-solving with an exciting user interface. Designed as a two-player logic challenge, the game tasks players with decoding a secret code while simulating the high-stakes experience of defusing a bomb. Featuring multiple difficulty levels, intuitive visual cues, and dynamic gameplay, this project highlights the creative application of programming concepts. The UI, inspired by a time bomb interface, includes elements like a keypad, LED indicators, and a tension-filled timer to immerse players in the experience. This game not only demonstrates technical skills in HTML, CSS, and JavaScript but also showcases an innovative approach to making learning fun and interactive.
